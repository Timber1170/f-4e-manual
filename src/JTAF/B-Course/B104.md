# B104 - Weapons Configuration Overview

#### Audience: Pilots unfamiliar with configuring the F-4E for weapons employment.
#### Given By: Instructor
#### Objective: Overview of the F-4Es Weapons Configuration System.

#### Class Activities

* Armament Top Level

    * Station Selection

    * AIM-9
